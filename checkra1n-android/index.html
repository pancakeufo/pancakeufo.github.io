Eseguire Checkra1n con Android Rooted

È possibile eseguire Checkra1n non solo su computer ma anche su device più piccoli: è il caso dei device Android con root

Giacomo

9 mag 2020 • 2 min read

Premessa

Questa è una guida per utenti esperti che hanno ben presente le procedure di rooting su Android e le hanno già eseguite.

Se stai cercando la guida a Checkra1n per Windows, Mac e Linux vai qui

Via TWRP

  • Scarica l'eseguibile di Checkra1n corretto per l'architettura del tuo device

    • Se non sai quale sia puoi controllarla con questo comando ADB:

      adb shell getprop ro.product.cpu.abi

  • Sposta l'eseguibile di Checkra1n nella directory /sdcard/Download

  • Riavvia il tuo device in modalità Recovery

  • Dal menu principale di TWRP spostati in Advanced → Terminal

  • Connetti il tuo iDevice al telefono Android
  • Digita il comando lsusb per verificare che il tuo iDevice sia stato riconosciuto
  • L'ID USB dovrebbe essere 05ac:12a8
  • Metti il tuo iDevice in modalità DFU
  • Digita nuovamente il comando lsusb e controlla che l'ID USB sia ora 05ac:1227
  • Spostati nella directory Download con il comando cd /sdcard/Download
  • Dai i permessi di esecuzione a Checkra1n con il comando chmod +x checkra1n
  • Avvia Checkra1n in modalità CLI tramite il comando ./checkra1n -c
  • Il processo di Jailbreak del tuo device dovrebbe iniziare

Via Terminale

  • Scarica l'eseguibile di Checkra1n corretto per l'architettura del tuo device

    • Se non sai quale sia puoi controllarla con questo comando ADB:

      adb shell getprop ro.product.cpu.abi

  • Sposta l'eseguibile di Checkra1n nella directory /data

  • Connetti il tuo iDevice al telefono Android

  • Apri il terminale e acquisisci i permessi di root con il comando su

  • Digita il comando lsusb per verificare che il tuo iDevice sia stato riconosciuto

  • L'ID USB dovrebbe essere 05ac:12a8

  • Metti il tuo iDevice in modalità DFU
  • Digita nuovamente il comando lsusb e controlla che l'ID USB sia ora 05ac:1227
  • Spostati nella directory /data
  • Dai i permessi di esecuzione a Checkra1n con il comando chmod +x checkra1n
  • Avvia Checkra1n in modalità CLI tramite il comando ./checkra1n -c
  • Il processo di Jailbreak del tuo device dovrebbe iniziare
