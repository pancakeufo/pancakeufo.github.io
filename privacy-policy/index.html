Policy

Nel pieno rispetto della privacy, giacomo.tips non fa uso di tracker, annunci o script traccianti nè raccoglie informazioni sensibili di alcun genere.

Il servizio Cloudflare, da noi utilizzato, potrebbe raccogliere alcuni dati minimali relativi ai log d'accesso per garantire la sicurezza del sito:

Cloudflare's __cfduid cookie is used to identify individual clients behind a shared IP address and apply security settings on a per-client basis. It does not correspond to any user ID in the web application and does not store any personally identifiable information.
