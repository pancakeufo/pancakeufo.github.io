Eseguire il Jailbreak con Checkra1n

Questa guida vi spiegherà come eseguire il jailbreak con Checkra1n, tool per le versioni di iOS da 12.0 a 14.8

Giacomo

30 dic 2021 • 5 min read

Introduzione

  • Questa guida vi spiegherà come eseguire il jailbreak con Checkra1n, tool semi-tethered per le versioni di iOS / iPadOS dalla 12.0 fino alla 14.4.2

Mac

  • Assicurati di avere un device compatibile
  • Scarica Checkra1n dal sito ufficiale facendo click su Get the beta Now e poi su Download for macOS
  • Se il sito dovesse avere problemi o risultare lento, usa web.telegram.org e cerca il canale @checkra1n per accedere ad un mirror del sito
  • Una volta scaricato il file aprilo con un doppio click e sposta l'app nella cartella Applicazioni, come suggerito dalla grafica
  • Sblocca ora il tuo device e connettilo al computer
  • Se viene richiesto di autorizzare il computer, tocca Consenti
  • Ora tieni premuto CTRL e fai click sull'app di Checkra1n, poi fai click su Apri
  • Fai click su Start

Premi Next e il tuo device verrà messo in recovery mode

  • Ora che il tuo iPhone è in modalità Recovery potrebbe apparire un avviso di questo genere, ignoralo
  • Ora arriva la parte un po' più complicata, mettere l'iPhone in modalità DFU.  Tranquillo, potrai fare quanti tenativi vorrai!
  • Se non sei pratico dell'inglese armati di Google Traduttore e traduci il riquadro che vi ho cerchiato nella foto che, ATTENZIONE, a seconda del device collegato mosterà istruzioni differenti
  • Premi ora Start ed esegui le azioni che ti indica l'applicazione
  • Se avrai fatto tutto correttamente apparirà una nuova schermata con una barra di avanzamento, altrimenti dovrai ripartire da capo
  • Quando apparirà la scritta All Done fai click su Done e spostati sul device che nel frattempo dovrebbe aver completato un riavvio
  • Apri l'applicazione checkra1n che è apparsa nella home del tuo dispositivo iOS:

  • Tocca Cydia nella sezione Install
  • Tocca Install e attendi
  • Complimenti! Hai eseguito correttamente il Jailbreak con Checkrain
  • Puoi ora usare Cydia per installare tweak, temi ed altri pacchetti
  • Ti consiglio di aggiungere queste sorgenti consigliate

Windows (Checkn1x)

  • Assicurati di avere un device compatibile
  • Scarica Checkn1x
  • Apri il sito ufficiale di Balena Etcher
  • Esegui il download dell'ultima versione
  • Collega una chiavetta USB vuota
  • Apri Balena Etcher
  • Fai click su Select Image
  • Seleziona il file scaricato dal sito di Checkn1x
  • Controlla che al centro sia selezionata la tua chiavetta, puoi verificarlo facilmente dalla capacità espressa in GB
    • Nello screenshot qui sotto potete vedere che la mia è da 32 GB
  • Dovresti ora avere una schermata simile a questa:

  • Fai click su Flash e attendi
  • Il processo sarà terminato quando apparirà questa schermata:

  • Ora, entra nella modalità BIOS del tuo computer

  • Una volta entrato nel BIOS dobbiamo:

    • disabilitare il Secure Boot
    • cambiare l'ordine di boot impostando l'USB per prima

  • Se hai fatto tutto correttamente checkn1x verrà avviato al riavvio del computer:

  • Premi Alt ed F2 contemporaneamente sulla tastiera per avviare Checkra1n:
  • Sblocca ora il tuo device e connettilo al computer
  • Se viene richiesto di autorizzare il computer, tocca Consenti
  • Usa le frecce sulla tastiera e il tasto invio per navigare
  • Spostati su Start e dai invio
  • Segui le istruzioni a schermo per mettere l'iPhone in modalità DFU, il programma farà il resto
  • Se non sei pratico dell'inglese armati di Google Traduttore
  • Quando apparirà la scritta All Done spostati sul device che nel frattempo dovrebbe aver completato un riavvio
  • Apri l'applicazione checkra1n che è apparsa nella home del tuo dispositivo iOS:

  • Tocca Cydia nella sezione Install
  • Tocca Install e attendi
  • Complimenti! Hai eseguito correttamente il Jailbreak con Checkrain
  • Puoi ora usare Cydia per installare tweak, temi ed altri pacchetti
  • Ti consiglio di aggiungere queste sorgenti consigliate

Linux

  • Assicurati di avere un device compatibile
  • Apri questa pagina
  • Scarica la versione di Checkra1n corretta per la tua architettura (solitamente x86_64)
  • Se vuoi esserne sicuro dai questo comando:

uname -a

  • Ecco per esempo il mio output, indicante che la mia architettura è x86_64:

Linux giacomo-tips 4.15.0-76-generic #86-Ubuntu SMP Mon Apr 6 15:10:11 UTC 2020 x86_64 x86_64 x86_64 GNU/Linux

  • Ora che hai scaricato l'eseguibile corretto di Checkra1n, spostalo sulla tua scrivania
  • Rinominalo in checkra1n
  • Apri ora un terminale e dai questi due comandi:

cd Desktop
chmod +x checkra1n

  • Puoi ora avviare checkra1n con questo comando:

sudo ./checkra1n

  • Checkra1n è ora in esecuzione:
  • Sblocca ora il tuo device e connettilo al computer
  • Se viene richiesto di autorizzare il computer, tocca Consenti
  • Usa le frecce sulla tastiera e il tasto invio per navigare
  • Spostati su Start e dai invio
  • Segui le istruzioni a schermo per mettere l'iPhone in modalità DFU, il programma farà il resto
    • Se non sei pratico dell'inglese armati di Google Traduttore
  • Quando apparirà la scritta All Done spostati sul device che nel frattempo dovrebbe aver completato un riavvio
  • Apri l'applicazione checkra1n che è apparsa nella home del tuo dispositivo iOS

  • Tocca Cydia nella sezione Install
  • Tocca Install e attendi
