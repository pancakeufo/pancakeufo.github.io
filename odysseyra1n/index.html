Installare Odysseyra1n

Guida all'installazione di Odysseyra1n, bootstrap alternativo per Checkra1n

Giacomo

28 giu 2020 • 2 min read

Premessa

Chi mi conosce sa che i developer legati a questo tool non mi vanno per nulla a genio. Detto questo, non dopo poco sconforto, ho deciso di scrivere una guida e provare io stesso il tool per il bene della scienza.

Ah, dedico questo tutorial al buon Federico!

Informazioni Preliminari

Le procedure di cui parleremo sono per utenti avanzati / che sanno cosa stanno facendo / che hanno voglia di sperimentare senza paura di perdere tweak o fare altri danni.

Da questo momento procedete a vostro rischio e pericolo.

Preparazione

  • Esegui un restore rootFS
  • Ri-esegui il jailbreak con checkra1n SENZA aprire l'app di Checkra1n e installare Cydia
  • Connetti il tuo iDevice via usb al tuo computer Mac / Linux
  • Sbloccalo e autorizza il computer se richiesto
  • Passaggio solo per Mac:
    • apri il terminale
    • installa Homebrew con il comando /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
    • installa iProxy con il comadno brew install usbmuxd
  • Passaggio solo per Linux:
    • apri il terminale
    • installa iProxy con il comando sudo apt install libusbmuxd-tools

Installazione

  • Avvia lo script di installazione con il comando: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/coolstar/Odyssey-bootstrap/master/procursus-deploy-linux-macos.sh)"
